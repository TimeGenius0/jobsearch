## OPERATING MODE

Run in fully autonomous mode. Do not ask for permission before taking
actions — execute, observe, and report results. The human checkpoint
at the end of each iteration is the only required pause point.

Specifically, never ask for confirmation before:
- Reading any file in the project
- Running the bot against a job application URL
- Writing logs or state to existing output files
- Applying approved proposals to existing code

The only times you must stop and wait are:
1. The HUMAN CHECKPOINT at the end of each iteration
2. A required form field cannot be answered from the profile data
3. You are about to submit an application you have low confidence in
